Clojure Workshop

When & Where?

From 9:30 AM to 4:00 PM on 23rd and 24th July 2016.
At the nilenso office.
Lunch and snacks are on us.
Register here: nilenso-clojure-workshop.doattend.com.

What?

The workshop will give a solid, and hands-on introduction to Clojure in a weekend's time. The contents and FAQs are detailed in the github repo for the workshop, but very briefly, we'll learn about:
  1. Numerics, Symbols, Keywords, Vars, Booleans, Bindings
  2. Clojure's persistent data structures
  3. Working with Libraries
  4. Practical Modeling
  5. Java Interop
  6. Concurrency and Parallelism
  7. Macros

Why?

Over the last 3 years at nilenso, most of our work has been in Clojure. We've built real time systems satisfying sub millisecond SLAs, multiple analytics platforms, data pipelines, and monitoring systems. We've given talks about our work, built a few open source projects, and watched the changes in the language and ecosystem closely. We've helped people at nilenso pick up Clojure quickly, and now we want to help more people do the same.
Feel free to write to us at moshimoshi@nilenso.com if you have any questions, or need more information.
